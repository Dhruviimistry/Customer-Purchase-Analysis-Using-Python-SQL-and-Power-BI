# Customer Data Analysis Using Python, SQL & Power BI

## Project Overview
This project demonstrates an end-to-end data analysis workflow using Python, SQL, and Power BI.  
It covers data cleaning, analysis, and visualization to understand customer behavior and business trends.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL (Data querying, joins, aggregations)
- Power BI (Interactive dashboards and reports)
- Excel (Data source)

## Topics Covered

### Python
- Data loading
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Handling missing values
- Data transformation

### SQL
- SELECT, WHERE, ORDER BY
- GROUP BY and HAVING
- Joins
- Subqueries
- Business-focused analytical queries

### Power BI
- Data modeling
- Measures and calculated columns
- Interactive dashboards
- Charts and filters

## Project Workflow
1. Collected and prepared raw data
2. Cleaned and analyzed data using Python
3. Performed data analysis using SQL
4. Built interactive dashboards in Power BI
5. Generated insights for decision-making

## Key Insights
- Customer purchasing patterns
- Sales and revenue trends
- Product and category performance
- Customer segmentation insights

## Purpose
This project is created to strengthen practical knowledge of data analytics tools and to showcase skills in Python, SQL, and Power BI.
